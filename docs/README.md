# Catrare Tech Website

カトレアテックの GitHub Pages 向けシンプルな1ページWebサイトです。

## ファイル構成

```text
.
├── index.html
├── README.md
└── assets/
    ├── style.css
    ├── logo.png
    ├── favicon.png
    └── apple-touch-icon.png
```

## GitHub Pagesで公開する手順

1. GitHubでリポジトリを作成します。
   - 例: `catrare-tech`
2. このZIPを展開し、ファイル一式をリポジトリ直下に配置します。
3. GitHubにpushします。
4. GitHubの `Settings` → `Pages` を開きます。
5. `Build and deployment` の Source を `Deploy from a branch` にします。
6. Branch を `main`、Folder を `/ (root)` にして保存します。
